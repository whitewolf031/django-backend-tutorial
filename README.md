React yordamida Full Stack loyiha yaratish

Ushbu qo‘llanmada Vite orqali React loyiha yaratish, o‘rnatish va ishga tushirish bosqichlari keltirilgan.

1. React o‘rnatish (Vite orqali)

Yangi React loyiha yaratish uchun quyidagi buyruqni terminalda ishga tushiring:

npm create vite@latest frontend -- --template react

Bu buyruq:

frontend nomli papka yaratadi

React template asosida loyiha generatsiya qiladi

2. Yaratilgan loyiha papkasiga kirish
cd frontend
3. Kerakli kutubxonalarni o‘rnatish
npm install

Bu buyruq loyiha uchun zarur barcha paketlarni (node_modules) o‘rnatadi.

4. Loyihani ishga tushirish (Development mode)
npm run dev

Shundan so‘ng terminalda lokal manzil chiqadi. Odatda:

http://localhost:5173/

Brauzer orqali shu manzilga kirib loyihani ko‘rishingiz mumkin.
