# React yordamida Full Stack loyiha yaratish

Ushbu qo‘llanmada React yordamida loyiha yaratish va ishga tushirish bosqichlari keltirilgan.

## 1. React o‘rnatish (Vite orqali)

Yangi React loyiha yaratish uchun quyidagi buyruqni terminalda ishga tushiring:

```bash
npm create vite@latest frontend

2. Yaratilgan loyiha papkasiga kirish
cd frontend

3. Kerakli kutubxonalarni o‘rnatish
npm install

Bu buyruq loyiha uchun zarur barcha paketlarni o‘rnatadi.

4. Loyihani ishga tushirish
npm run dev

Shundan so‘ng terminalda ko‘rsatilgan lokal manzil orqali loyihani brauzerda ochishingiz mumkin (odatda http://localhost:5173).
